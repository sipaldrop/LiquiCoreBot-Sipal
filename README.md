## ÔøΩ Instalasi (Cara Pasang)

Ikuti langkah-langkah ini perlahan untuk pemula:

### 1. Download Bot
Buka terminal (Command Prompt atau PowerShell) dan ketik perintah ini:

```bash
git clone https://github.com/sipaldrop/LiquiCoreBot-Sipal.git
cd LiquiCoreBot-Sipal
```

### 2. Install Aplikasi Pendukung
Masih di terminal, ketik perintah ini untuk menginstall semua kebutuhan bot:

```bash
npm install
```
*Tunggu sampai proses selesai. Mungkin butuh beberapa menit.*

## ‚öôÔ∏è Konfigurasi (Pengaturan)

### 1. Buat File Akun (`accounts.json`)
Kamu perlu membuat file untuk menyimpan data akunmu.
1.  Cari file bernama `accounts_example.json` di folder bot.
2.  Ubah namanya (rename) menjadi `accounts.json`.
3.  Buka file tersebut dengan Notepad atau Text Editor.
4.  Isi data akunmu seperti format di bawah ini:

```json
[
  {
    "privateKey": "PRIVATE_KEY_METAMASK_MU",
    "proxy": "http://user:pass@ip:port"
  },
  {
    "privateKey": "PRIVATE_KEY_AKUN_KEDUA",
    "proxy": null
  }
]
```
> **Catatan:**
> - `privateKey`: Private key wallet Metamask kamu (Hati-hati jangan disebar!).
> - `proxy`: (Opsional) Jika pakai proxy, isi formatnya. Jika tidak, isi `null`.

### 2. Konfigurasi Lain
Tidak ada file `config.json` yang perlu diatur. Semua pengaturan default sudah tertanam otomatis di dalam bot.
